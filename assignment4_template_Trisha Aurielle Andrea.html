Topic of this work

How do COVID-19 vaccination rates correlate with COVID-19 case rates and mortality across different regions?

Data download

Downloading Dataset 1
Dataset link

https://www.kaggle.com/datasets/antgoldbloom/covid19-data-from-john-hopkins-university?resource=download

Description

Global COVID-19 data, including the number of confirmed cases, fatalities, and recoveries for different nations and regions, are included in this collection. It is a trustworthy resource for monitoring the global spread of the pandemic and is updated frequently.

Download Process

The dataset can be downloaded directly from Kaggle by using the provided link.

Downloading Dataset 3
Dataset link

https://www.kaggle.com/datasets/paultimothymooney/oxford-covid19-government-response-tracker

Description

This dataset documents national governments’ reactions to COVID-19, including travel bans, mask requirements, and lockdowns. It contains daily information on the occurrence and severity of governmental actions.

Download Process

The dataset can be downloaded directly from Kaggle by using the provided link, and then i converted this xlsx file into csv format.

Data Usage

Analyzing the effects of government policy on vaccination rates, case counts, and mortality requires this data. For a more thorough analysis, it can be paired with COVID-19 case data and vaccination data.
